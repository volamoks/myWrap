# Инструкция по редактированию

Этот сайт можно редактировать без программиста! Все тексты, фото и темы находятся в одном файле.

## Где редактировать?
Откройте файл `public/config.json` в любом текстовом редакторе (Блокнот, VS Code, TextEdit).

## Структура файла
Файл состоит из блоков `{ ... }`, каждый блок — это один "слайд" истории.

### Примеры слайдов

**1. Титульный слайд (welcome)**
```json
{
    "type": "welcome",
    "title": "Привет!", 
    "subtitle": "Текст под заголовком",
    "theme": "red" 
}
```
* `theme`: цвет фона. Варианты: `red`, `blue`, `green`, `yellow`, `purple`.

**2. Слайд с цифрой (stat)**
```json
{
    "type": "stat",
    "title": "Свидания",
    "value": "42",
    "description": "Описание под цифрой",
    "theme": "blue"
}
```

**3. Слайд с фото (photo-grid)**
```json
{
    "type": "photo-grid",
    "title": "Наши фото",
    "description": "Подпись внизу",
    "images": [
        "https://ссылка-на-фото-1.jpg",
        "https://ссылка-на-фото-2.jpg"
    ],
    "theme": "yellow"
}
```
* `images`: список ссылок на фото. Можно использовать ссылки из интернета или положить свои фото в папку `public/photos` и писать путь так: `/photos/my-photo.jpg`.

**4. Итоги (summary)**
```json
{
    "type": "summary",
    "title": "Наш Год",
    "stats": [
        { "label": "Свиданий", "value": "42" },
        { "label": "Звонков", "value": "840ч" }
    ],
    "theme": "purple"
}
```

## Как сохранить?
Просто сохраните файл (`Ctrl+S` или `Cmd+S`) и обновите страницу в браузере. Изменения появятся сразу!
